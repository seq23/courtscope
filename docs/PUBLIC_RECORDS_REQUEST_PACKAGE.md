# CourtScope Public Records Request Package

## Draft request

To the Public Records Request Coordinator, Shelby County Criminal Court Clerk's Office:

CourtScope requests an electronic, machine-readable extract of public criminal-court case and sentencing records for Shelby County Criminal Court for the most recent eight complete calendar years. CSV, database export, JSON, or another native structured format is preferred over PDF. CourtScope is an open-source civic research project examining aggregate sentencing patterns. It will not publish defendant names and will not issue individual-case fairness judgments.

Please provide, where maintained and legally disclosable:

- stable case identifier and public case number;
- indictment number and count number;
- court, division, docket, and courtroom assignment;
- assigned judge, disposition judge, and sentencing judge;
- charge statute/code, description, grade/class, and enhancements;
- filing, disposition, and sentencing dates;
- disposition and plea-versus-trial indicator;
- incarceration imposed, suspended, and to serve;
- probation, community corrections, fines, restitution, credit for time served, and concurrent/consecutive structure;
- offender range, prior-record classification, or sentencing worksheet fields;
- race, ethnicity, age/date-of-birth-derived age, and gender where maintained and legally disclosable;
- record creation, update, correction, and supersession indicators;
- data dictionary, code tables, schema, and field definitions.

Please exclude sealed, expunged, or otherwise non-public records. If the full request is burdensome, please provide a representative sample extract and a cost/time estimate before producing the full dataset. Please identify fields that are unavailable, exempt, or not maintained, and whether demographic fields are self-reported, observed, imported, or derived.

## Preferred delivery

- encrypted electronic delivery or approved secure download;
- one file per year if schema changed materially;
- native encoding and unmodified source values;
- a checksum if the source system supports one.
