# Shelby County Criminal Court Source Map

## Status

Phase 4 source reconnaissance is complete. A verified public bulk case dataset was not available in the build environment. The official public-records request lane is therefore the primary bulk-acquisition route.

| Source | Authority | Intended use | Automation state |
|---|---|---|---|
| Criminal Court Clerk website | Official | Clerk policy, contacts, access notices | Reference only |
| Criminal case information access | Official | Case-level verification if authorized | Access review required |
| Clerk public-records process | Official | Machine-readable historical bulk extract | Request package ready |
| Shelby County Criminal Court page | Official | Court identity and public instructions | Reference only |
| Tennessee AOC judge directory | Official | Current judge identity seed | Adapter planned; history incomplete |
| Shelby County Election Commission | Official | Election and term corroboration | Later phase |

## Governing rule

CourtScope will not automate a portal until its endpoint, access terms, rate limits, authentication model, available fields, and permitted use are documented. No CAPTCHA bypass, credential sharing, or access-control circumvention is allowed.
