# Data Pipeline in Plain English

1. Register an official source.
2. Download or receive source material.
3. Save evidence and checksum it.
4. Convert records to CourtScope fields.
5. Resolve court, division, judge, offense, sentence, and supported demographics.
6. Skip questionable records instead of guessing.
7. Run privacy and quality checks.
8. Recalculate aggregate models.
9. Publish only when all gates pass.
10. Generate receipts and public data packages.
