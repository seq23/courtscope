# Source Access and Terms

## Allowed acquisition lanes

1. Official machine-readable files delivered by the records custodian.
2. Official public APIs or downloads whose terms permit the intended use.
3. Authorized portal access after written access and rate-limit review.
4. Official documents for limited reconciliation when structured data are incomplete.

## Prohibited behavior

- bypassing login or CAPTCHA controls;
- evading rate limits;
- using undocumented private endpoints without authorization;
- presenting unofficial mirrors as canonical evidence;
- inferring missing protected attributes from names, photographs, or neighborhoods.

## Current decision

The prepared Tennessee Public Records Act request is the canonical bulk-data acquisition path until an official structured portal or API is verified.
